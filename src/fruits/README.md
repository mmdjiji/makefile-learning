# fruits
A test project for [mmdjiji/makefile-learning](https://github.com/mmdjiji/makefile-learning).

## Deploy without install
```sh
make
```

## Run
```sh
./fruits
```

## Deploy and install
```sh
make && make install
```
After installation, you can use `fruits` command everywhere.

## Clean
```sh
make clean
```
Clean the object files.

## Uninstall
```sh
make uninstall
```
